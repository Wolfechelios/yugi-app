## [1.0.0] - 2024-10-15

### Added
- Initial release of DuelVault
- Next.js 15 with App Router
- TypeScript implementation
- Tailwind CSS with shadcn/ui components
- Prisma ORM with SQLite database
- Tesseract.js OCR integration
- Socket.IO real-time communication
- Card scanning API endpoints
- Collection management system
- User authentication foundation
- Development and production configurations

### Features
- **Card Scanning**: Upload and scan Yu-Gi-Oh! cards with OCR
- **Manual Entry**: Complete card information input system
- **Collection Management**: Organize cards in custom collections
- **Search & Filter**: Find cards by various attributes
- **Statistics Dashboard**: Track collection metrics
- **Batch Processing**: Scan multiple cards simultaneously
- **JSON Import**: Support for official card data formats
- **Real-time Updates**: Live scanning progress and notifications

### Technical Stack
- Frontend: Next.js 15, React 19, TypeScript 5
- Styling: Tailwind CSS 4, shadcn/ui
- Database: Prisma ORM, SQLite
- OCR: Tesseract.js with image preprocessing
- Real-time: Socket.IO
- Development: ESLint, Prettier, Nodemon
